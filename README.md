# analisis-estudiantes.R
Análisis de Desempeño Académico de Estudiantes
# Descripcion
análisis exploratorio de datos (EDA) realizado en R, utilizando un dataset de desempeño académico estudiantil.
# Objetivo
El objetivo es analizar la relación entre horas de estudio, actividades extracurriculares y rendimiento académico, aplicando técnicas básicas de visualización y manipulación de datos.
El proyecto está orientado a reforzar habilidades en R para análisis de datos, especialmente en el uso de funciones base y librerías como ggplot2 y dplyr.
